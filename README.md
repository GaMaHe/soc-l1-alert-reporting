This repository documents hands-on experience simulating the role of a **Security Operations Center (SOC) Level 1 Analyst**, focused on alert triage, analysis, documentation, and initial incident response using a simulated SIEM dashboard.

The objective of this project is to demonstrate practical SOC L1 workflows, analytical thinking, and structured incident reporting rather than tool-specific expertise.

## üîç Scope of the Project

This project simulates real-world SOC Level 1 responsibilities, including:

- Monitoring and reviewing security alerts from a SIEM-like dashboard

- Investigating suspicious activity using available evidence

- Documenting findings in a clear and structured manner

## üìå Methodology

The investigation process followed for each alert:

1. Review alert details and severity  
2. Analyze available context and indicators  
3. Assess likelihood of malicious activity and potential impact 
4. Classify the alert (false positive / suspicious / incident)  
5. Document findings and reasoning  
6. Decide whether escalation is required  

Detailed methodology is documented in the `docs/` directory.
